# OpenTypeless v1.1.48

This release focuses on making OpenTypeless feel faster, clearer, and more reliable across the everyday voice workflow.

## Highlights

- **Typeless-style hotkeys**
  - macOS dictation now defaults to `Fn`.
  - Windows dictation now defaults to `Right Alt`.
  - Ask Anything has its own shortcut: `Fn+Space` on macOS and `Right Alt+Space` on Windows.
  - Linux keeps `Ctrl+/` for dictation and `Ctrl+.` for Ask Anything because Right Alt global handling is less reliable across Linux desktop environments.

- **Ask Anything feels lighter**
  - Ask now records through the capsule instead of showing a second recording panel.
  - Results appear in a compact floating note.
  - The answer can be copied directly.
  - Clicking outside the note dismisses it.

- **Cleaner account and upgrade experience**
  - Upgrade now uses the same quiet product UI as the rest of the app.
  - Pricing, plan status, and benefits are easier to scan without adding extra setup choices.

- **Better capsule flow**
  - The capsule now has clearer preparing, recording, transcribing, polishing, Ask recording, Ask thinking, and output states.
  - Idle capsule visibility can be controlled from General settings, the capsule context menu, or the tray.

- **Writing styles and scenes**
  - Added polish styles: Minimal, Clean, Structured, and Professional.
  - Added built-in scenes, local custom scenes, scene activation metadata, and import/export.
  - Scenes make reusable writing behavior easier to manage without turning Settings into a heavy editor.

- **Selected text workflows**
  - Ask and polish can use selected text as context.
  - Selected text is truncated before AI requests for privacy and cost control.
  - The app separates destructive edits from non-destructive questions more carefully.

- **Dictionary and correction rules**
  - The custom dictionary is now paired with local correction rules.
  - This helps fix recurring transcription mistakes and domain-specific phrases.

- **More reliable hotkeys and output**
  - Hotkey registration now has clearer status, collision checks, retry behavior, and rollback.
  - Output strategies include keyboard simulation, clipboard paste, copy-only fallback, and Windows SendInput.
  - Clipboard restore and output diagnostics are safer and easier to understand.

- **Local and BYOK improvements**
  - Added macOS Apple Speech as a built-in local speech provider.
  - Improved custom Whisper/self-hosted STT diagnostics.
  - BYOK secrets are stored in the OS credential vault where available.

## Notes

- Windows builds may still be unsigned depending on release secrets.
- macOS and Linux release artifacts are expected to use the configured signing keys.
- English and Chinese UI copy are complete for this release. Other bundled locale files have full key coverage, but some newer advanced feature copy may still display English fallback text.
- Cloud remains optional. BYOK and local/self-hosted paths continue to work without OpenTypeless cloud services.
